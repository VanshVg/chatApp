
# Chat-App

Have built basic chat web app using socket.io, express and Node Js.
- The project uses:
    * Node.Js
    * Express.Js
    * socket.io
    
## Installation
To install dependencies/packages

```bash
  npm install
```

## Deployment
To deploy this project
```bash
  npm run dev
```
